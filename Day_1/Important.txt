JDK(Java Development Kit) includes JRE(Java Runtime Environment) and Development tools ; and JRE includes JVM(Java Virtual Machine) and Libraries . 

    Question : How is my code running ? 
        1. Compilation : 
            SourceCode (.java) ---> Compiler ---> byte code (.class)    // you can run the byte code on any system until and unless JDK is installed in that machine . That's why it is a very portable language .

        2. Execution :  
            Byte Code (.class) ---> JVM ---> Native Code
